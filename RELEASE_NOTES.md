# MemoryLane v0.13.11

Patch release focused on workflow improvements and CLI access.

## What's Changed

- **Standalone CLI package** - Added `@deusxmachina-dev/memorylane-cli` for querying activity history from AI agents without running the Electron app
- **Configurable capture hotkey** - Added configurable start/stop capture hotkey support
- **Runtime decoupling cleanup** - Decoupled core modules from direct Electron imports for safer non-Electron execution paths
- **Path/config hardening updates** - Isolated dev/prod config-state paths and cleaned up capture setting naming/legacy aliases
